# React Todo
React.js와 CSS를 사용하여 만든 todo 어플리케이션 
- input : text input 
- Btn : all, active, completed 된 todo들만 선택 가능
- Btn : all Clear : completed된 todo 삭제
---
# 추가 예정
- checkbox : toggle logic 수정 후 등록 예정
- storage : localstorage 구현 예정
- view : 모바일에 맞춰 수정 예정
